The clinical research repository is a file based repository - capable of holding any type of 'digital object' - be it a data set, a document, a computer program, or a media file. The secure infrastructure for file storage is provided by TSD, an established secure data infrastructure that is part of the University of Oslo.
ECRIN liaises with both data providers, for the transfer of digital objects to the repository, and data requesters, browsing the objects under managed access. Both processes are governed by formal agreements, one for the data providers, and another for the secondary users. ECRIN uses a Repository Management System, developed in-house, to manage and record this activity. The system developed so far is intended to provide and support: the management of repository content and its associated metadata; the transfer of data objects into the secure infrastructure; the collection of details of data contributors and their requirements; the quality control processes applied to uploaded data objects; the handling of data requests, in particular the responses of the repository in answering data requests; the usage of all types of stored data objects; the repositoryâ€™s performance against pre-agreed quality criteria.
